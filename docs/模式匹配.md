# 模式匹配
## ref 关键字
ref是一个关键字，用于模式匹配。专门在通过let绑定或match分支解构结构体，元组等复合类型时，***创建一个对字段的引用，而不是移动该字段的所有权***
```rust
use std::io::BufReader;
use std::net::TcpStream;
let stream = TcpListener::bind("xx:xx").unwrap();
let buf = BufReader::new(stream);
let request = buf.lines().next().unwrap().unwrap();

//如果下面不使用ref关键字，request会被移动到match分支中，导致request被移动
let (status, filename) = match request {
    ref line if line == "GET / HTTP/1.1" => ("200", "/"),
    _ => ("404", "/404.html"),
};
```
* 用途：仅在模式中用于借用数据， 防止所有权被移动
* 非匹配部分： ref只是绑定方式说明符，它不影响值是否匹配成功
* 类似mut： ref的工作方式类似于mut， 它为模式绑定增加了表达力
