# 变量和可变性

## 变量和可变性
变量默认是不可变的。 只有显示的声明为可变变量时 才能改变它的值。 这样编译器久知道哪些变量需要追踪其变化，哪些不需要追踪。
如果要使某个变量时可变的， 需要在其变量名前面加上mut，使其变为可变

## 常量
常量不只是默认不可变，并且它们永远不可变。常量可以在任何作用域中声明，包括全局作用域
常量只能被设置为常量表达式，不能是那些只能在运行时计算出来的值。(常量是编译时运算) 
声明时要用到const关键字，而不是let。 并且必须标注类型。Rust 对常量的命名约定是全部大写，并用下划线分隔单词

## 遮蔽
遮蔽是创建了一个新的同名变量， 而mut 则是改变原油变量。
遮蔽所创建出的新变量的类型可以与原有变量不同


常量是否能被遮蔽？
  1. const 常量无法被遮蔽(常量变量均无法遮蔽)
  2. 变量只能被变量遮蔽。
  3. 同一作用域不能有同名的常量， 同一作用域常量和变量不能同名
